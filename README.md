# Set up

1. Setup Prettier on VScode
2. Install Yarn
3. Create `.env` file with `DATABASE_URL`
4. `yarn`
5. `yarn dev`

# Database

Online Heroku Postgresql server

After modifying the database, run:

`npx prisma generate`

# Fix Typescript error relating to `prisma.ts`

run `yarn add -D @prisma/client`
